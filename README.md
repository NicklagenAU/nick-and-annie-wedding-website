# Nicholas and Annie Wedding Website

This is a personal wedding website for my Fiance and I. The site includes a custom landing page with interactive sections for the welcome message, venue information, dress code, travel accomadations, and registry.

## Features

- Custom HTML, CSS, and JavaScript website
- Interactive section buttons that display selected wedding details
- Hosted images through Amazon S3
- Website served from an AWS EC2 instance
- Custom domain configured through Amazon Route 53
- External links for venue information and wedding registry

## Technologies Used

- HTML
- CSS
- JavaScript
- AWS EC2
- Amazon S3
- Amazon Route 53

## Purpose

The purpose of this project is to create a simple, elegant, and easy-to-use wedding website where guests can quickly find important wedding information.
